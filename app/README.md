# 工布江达低空智慧文旅平台演示系统

这是一个基于 React + Vite + TypeScript 的演示系统，用于展示低空无人机文旅应用。

## 功能特性

- **指挥中心大屏**：可视化无人机航线、禁飞区和实时状态
- **游客端租赁页**：产品卡片展示和地图预览
- **演示剧本**：Debug 面板触发特定业务场景

## 安装和运行

```bash
npm install
npm run dev
```

打开 http://localhost:5173 查看应用。

## 项目结构

- `src/types/` - 类型定义
- `src/data/` - Mock 数据
- `src/components/` - React 组件
