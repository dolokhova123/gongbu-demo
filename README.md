# 工布江达无人机文旅平台

一个完整的无人机旅游平台，包含指挥中心监控系统和游客端应用。

## 功能特性

### 🚁 指挥中心
- 实时运营数据监控
- 无人机状态跟踪
- 景区收入统计
- 航线规划管理

### 📱 游客端应用
- 无人机航线预订
- 无人机租赁服务（APP远程/现场租赁）
- **AI生成内容服务**：
  - **真人真景合成**：上传个人照片+景区打卡点照片，AI智能合成专属景区写真
  - **虚拟形象生成**：上传个人照片，AI创建3D虚拟分身，可在虚拟景区中漫游
- 照片上传和AI处理

### ⚙️ 系统设置
- 主题切换（深色/浅色）
- 语言设置
- 通知管理
- 系统信息查看

### ❓ 帮助中心
- 详细使用指南
- 常见问题解答
- 联系方式和客服信息

## AI功能详解

### 🎨 真人真景合成
1. **上传个人照片**：选择正面清晰的个人照片
2. **选择景区打卡点**：为每个景区选择对应的打卡点照片
   - **巴松措**：扎西岛、扎拉沟瀑布
   - **哈巴错**：冰川之眼、崩嘎村
   - **念朗温泉**：田园风光、温泉区
3. **AI智能处理**：系统自动分析面部特征和景区场景
4. **生成结果**：获得个性化景区写真和数字收藏品

### 🎭 虚拟形象生成
1. **上传个人照片**：用于生成3D虚拟形象的基础
2. **选择虚拟场景**：根据景区特色提供不同场景选项
   - **巴松措**：湖边禅修、山顶观景
   - **哈巴错**：冰川探险、高原牧场
   - **念朗温泉**：田园采风、温泉疗养
3. **AI建模**：基于照片创建栩栩如生的3D虚拟分身
4. **沉浸体验**：可在虚拟景区中自由漫游和互动

### 📊 处理流程
- **文件验证**：支持图片格式，最大5MB
- **AI处理**：模拟3秒处理时间（实际应用中可连接真实AI服务）
- **结果展示**：高清图片展示，支持下载和分享
- **预约记录**：处理结果保存在"我的预约"中

## 主要组件说明

### OperatorDashboard
指挥中心主界面，显示系统总览、统计数据和地图视图。

### TouristApp
游客端应用，包含服务选择、预订和AI内容生成功能。

### Settings
系统设置页面，支持主题、语言和通知配置。

### Help
帮助中心，提供使用指南、FAQ和联系方式。

## 数据结构

- **服务类型**: 无人机租赁、生成式内容
- **航线数据**: 三个主要景区（巴松措、哈巴错、念朗）
- **统计信息**: 收入、用户、订单等运营数据

## 响应式设计

平台支持多种屏幕尺寸，采用移动优先的响应式设计，确保在手机、平板和桌面设备上都能良好显示。
